# Meow Belle — product photos

The 13 product photos in `images/products/` were extracted from Z's Excel catalogue
(`Product catalog Excel Final.xlsx`), matched to each product by reading the packaging,
and normalised to a square 1000×1000 white frame for a consistent catalog look.
(Two competitor-watermarked images from the Excel — "JJPetShop" and
"meowmeowpawshopbd.com" — were deliberately skipped.)

The site loads each product's photo by the filename below. To replace one with a better
shot, drop a new square image in with the same filename (keep it `.png`).

# Meow Belle — product photos

Drop your real photos into `images/products/` using these **exact filenames**. The site
shows a branded placeholder tile until a matching file exists, then swaps in your photo
automatically — no code changes needed.

| Filename | Product | Best size |
|---|---|---|
| `belles-daily.jpg` | Belle's Daily (Chicken & Salmon dry food) | 1000 × 1000 px, square |
| `ocean-catch.jpg` | Ocean Catch (Whitefish & Tuna dry food) | 1000 × 1000 px, square |
| `little-belle.jpg` | Little Belle (Kitten formula) | 1000 × 1000 px, square |
| `purr-wet-pack.jpg` | Purr Wet Pack (24 gravy pouches) | 1000 × 1000 px, square |
| `belle-bites.jpg` | Belle Bites (freeze-dried treats) | 1000 × 1000 px, square |
| `full-bowl-bundle.jpg` | The Full Bowl Bundle | 1000 × 1000 px, square |

**Tips**
- Square images look best on cards and the product gallery.
- Shoot on a plain light or electric-blue background to match the brand.
- Prefer `.jpg` for photos. If you use `.png` or `.webp`, also update the `image:` path
  for that product in `assets/js/products.js` (just change the file extension).
- The `hero.png` at the project root is already wired in as the homepage hero.
